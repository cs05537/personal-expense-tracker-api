# Personal Expense Tracker API 💰

Ένα RESTful API γραμμένο σε Python, σχεδιασμένο για τη διαχείριση προσωπικών εξόδων. 

Αυτό το project αναπτύχθηκε για την κατανόηση της δημιουργίας web APIs, τη διαχείριση βάσεων δεδομένων με Python και τον χειρισμό HTTP requests (GET, POST).

## Τεχνολογίες που χρησιμοποιήθηκαν
* **Python 3**
* **FastAPI:** Για τη δημιουργία των API endpoints.
* **SQLite:** Ενσωματωμένη βάση δεδομένων για την αποθήκευση των εξόδων.
* **Uvicorn:** ASGI server για την εκτέλεση της εφαρμογής.
